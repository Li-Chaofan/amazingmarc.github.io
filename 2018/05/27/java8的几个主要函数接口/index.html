java8的几个主要函数接口

发表于 2018-05-27 | 分类于 java8

java8 中在java.util.function中引入了很多函数接口，我们详对其中几个主要接口做一下说明。包括 Supplier, Consumer, Function, BinaryOperator, Predicatey以及 BiSupplier,BiConsumer,BiFunction,BiPredicate。

顾名思义

  • supplier : 提供者

  • consumer : 消费者

  • function : 函数

  • operation : 运算符

  • binary : 二元 （二元一次方程）

  • predicate : 断言

总结

总结一下，Supplier不接受任何参数，返回一个值。Consumer 接收一个参数，处理，不返回任何值。Function，接收一个参数，处理，返回一个值。BinaryOperator接收两个参数，处理，返回相同类型的值。Predicate 接收一个参数，返回true or false。 BiSupplier,BiConsumer,BiFunction,BiPredicate 均是在原有基础上多接收一个参数。
